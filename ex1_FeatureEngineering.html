Hng dn 7a: Xy dng c trng (Feature Construction)

Feature Construction l qu trnh to ra cc c trng mi t nhng c trng hin c trong tp d liu,
nhm gip m hnh hc my hiu r hn v cu trc d liu v ci thin hiu sut d on.

Qu trnh ny thng bao gm:

  • Kt hp (combining) cc c trng hin c to ra c trng mi
  • Bin i (transforming) c trng (v d: log, cn bc hai, chun ha, chun ha min-max)
  • Rt trch thng tin (extracting information) t c trng dng ngy thng, chui k t, hoc danh mc
  • M ha (encoding) c trng dng phn loi (categorical features) thnh dng s

Mc tiu cui cng l tng kh nng m hnh ha mi quan h tim n trong d liu, gip cc thut ton hc my hot ng hiu qu hn.

Mc ch:
on code ny dng chia d liu thnh hai phn: tp hun luyn (training set) v tp kim tra (testing set) nhm phc v cho qu trnh xy dng v nh gi m hnh hc my.

C th:

  • train_test_split() l hm trong th vin sklearn.model_selection gip tch d liu.
  • X l tp cc bin c lp (c trng u vo), cn y l bin mc tiu (nhn).
  • test_size=0.3: 30% d liu c dng kim tra, 70% cn li dng hun luyn m hnh.
  • random_state=42: m bo vic chia d liu din ra ngu nhin nhng ti lp c (gip kt qu n nh khi chy li).
  • stratify=y: chia d liu theo t l ca cc lp trong bin mc tiu, trnh mt cn bng lp gia tp train v test.

Kt qu:
Sau khi chy, ta thu c 4 bin:

  • X_train, y_train: d liu hun luyn m hnh.
  • X_test, y_test: d liu kim tra chnh xc ca m hnh.

PCA

Chun ha d liu (Standardize the Data)

PCA (Phn tch thnh phn chnh Principal Component Analysis) b nh hng bi thang o ca d liu,
v vy, trc khi p dng PCA, ta cn a tt c cc c trng v cng mt thang o,
c th l c gi tr trung bnh (mean) = 0 v phng sai (variance) = 1.

Chng ta s s dng StandardScaler chun ha (standardize) cc c trng trong tp d liu.

In[]:
#Now lets apply 1-1 "StandardScaler" transformer
#1) import the module
from sklearn.preprocessing import StandardScaler

#2) define the model
scaler=StandardScaler()

#3) fit the model
scaler.fit(X_train)

#4) transform the data
X_train_ss = scaler.transform(X_train)

# note 3 and 4 could be combined like this:
# X_train_ss = SS.fit_transform(X_train)

Run PCA

#Now lets perfrom pca

Bin i d liu kim tra (Transform the Test Data)

Lu : bc ny, chng ta ch p dng cc b bin i (transformers) c hun luyn t d liu hun luyn bin i tp d liu kim tra.
Tuyt i khng s dng phng thc fit trn tp kim tra,
v vic s khin m hnh hc thm thng tin mi t d liu kim tra,
dn n r r d liu (data leakage) v kt qu nh gi khng chnh xc.

In[]:
#1- First apply the scaler that has been built based on the training data to scale the test data
X_test_ss = scaler.transform(X_test)
#2- Second apply the pca transformation that has been built based on the training data to transfer the scaled test data
pca_test = pca.transform(X_test_ss)

ICA

Cc bc thc hin

Cc bc trong phn ny rt ging vi quy trnh ca StandardScaler v PCA transformers.

Thc t, c nhiu phng php ICA (Independent Component Analysis) khc nhau,
nhng trong hng dn ny, chng ta s s dng thut ton Fast ICA c cung cp trong th vin sklearn.

In[]:
from sklearn.decomposition import FastICA
ica = FastICA(n_components=2)
ica_train = ica.fit_transform(X_train_ss)  # nb. we already did the scaling, above
In[]:
# Visualisation
df_ica_train = pd.DataFrame(data = ica_train, columns = ['ic1', 'ic2'])
df_ica_train['species']=y_train
sns.scatterplot(x='ic1', y='ic2', hue=df_ica_train['species'], data=df_ica_train);

on code ny trc quan ha d liu sau khi gim chiu bng Independent Component Analysis (ICA).
DataFrame df_ica_train c to vi hai thnh phn c lp ic1 v ic2, cng nhn species t tp hun luyn.
Lnh sns.scatterplot(...) v biu phn tn th hin cch cc mu d liu c phn tch theo tng loi hoa.
Nu cc cm im c xu hng tch bit nhau, iu cho thy ICA gip biu din d liu tt hn cho vic phn loi.

In[]:
# Performing classification using the ica-based transformed data
# Transform test data using ica
classifier.fit(ica_train, y_train)
ica_test = ica.transform(X_test_ss)
score = accuracy_score(classifier.predict(ica_test), y_test)
print('Accuracy after ICA transformation  = {:.2f}'.format(score))

Accuracy after ICA transformation  = 0.93

on code ny thc hin phn loi da trn d liu c chuyn i bng Independent Component Analysis (ICA).
C th:

  • M hnh classifier c hun luyn trn d liu hun luyn qua ICA (ica_train, y_train).
  • Tp kim tra X_test_ss cng c bin i bng cng m hnh ICA to ica_test.
  • Sau , m hnh d on nhn cho ica_test v tnh chnh xc (accuracy) bng accuracy_score.

Kt qu in ra cho bit mc chnh xc ca m hnh sau khi p dng ICA, gip nh gi hiu qu ca k thut gim chiu i vi hiu sut phn loi.
