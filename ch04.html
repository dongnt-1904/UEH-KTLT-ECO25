import numpy as np

my_arr = np.arange(1_000_000)
my_list = list(range(1_000_000))
# print(my_arr)
# print(my_list)

To ra hai cu trc d liu cha cng mt dy s t 0 n 999,999 nhng khc nhau v loi:

  • my_arr l mt mng NumPy (ndarray) c to bng np.arange(1_000_000).
  • my_list l mt danh sch Python thng thng c to bng list(range(1_000_000)).

Vic to song song hai i tng ny thng c dng so snh hiu sut gia NumPy v list ca Python trong cc php ton s hc hoc x l d liu ln.
NumPy thng nhanh v tit kim b nh hn nh c ti u bng C v h tr vectorization.

In[3]:
%timeit my_arr2 = my_arr * 2
%timeit my_list2 = [x * 2 for x in my_list]

1.8 ms  800 s per loop (mean  std. dev. of 7 runs, 1000 loops each)
61.6 ms  12.8 ms per loop (mean  std. dev. of 7 runs, 10 loops each)

S dng magic command %timeit ca Jupyter Notebook o thi gian thc thi trung bnh ca hai php nhn trn hai cu trc d liu khc nhau:

  • %timeit my_arr2 = my_arr * 2
    Nhn ton b phn t trong mng NumPy my_arr vi 2 (theo kiu vector ha).
    NumPy thc hin php nhn ny cc nhanh v c ti u mc thp (C-level operations).

  • %timeit my_list2 = [x * 2 for x in my_list]
    Nhn tng phn t trong danh sch Python my_list vi 2 bng list comprehension, chm hn v php nhn din ra tun t trong Python interpreter.

Kt qu cho thy NumPy nhanh hn rt nhiu ln so vi list khi thao tc vi d liu ln.

  • data + data: cng tng phn t tng ng gia hai mng data.
    Tc l mi gi tr trong data c nhn i (v cng chnh n).

    • position += step: cp nht v tr hin ti.
